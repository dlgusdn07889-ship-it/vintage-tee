# Vintage Tee Finder — 준비용 스타터

eBay 개발자 계정 승인을 기다리는 동안 준비하는 프로젝트입니다.

현재 포함:
- GitHub Pages 웹 대시보드
- 경매 / 즉시구매 필터
- 태그 / 사이즈 / 키워드 검색
- 샘플 매물 표시
- Telegram 연결 테스트
- eBay API 연결용 파일 구조
- GitHub Actions 자동 실행 틀

민감정보인 봇 토큰과 eBay Client Secret은 파일에 적지 말고
GitHub의 Settings → Secrets and variables → Actions에 저장하세요.
